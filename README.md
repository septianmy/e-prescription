## Installation 
```
We assumed you have installed Laravel 

1. Set your database, database name : db_test, database username : root, database password : ''
2. Copy file env.example to your .env
3. Run your terminal and run this command below
    1. composer install
    2. php artisan migrate:fresh --seed
    3. php artisan serve
4. Open the application with accesss 'http://localhost:8000'
```
